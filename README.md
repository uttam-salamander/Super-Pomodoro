# Super Pomodoro

Super Pomodoro is a cross-platform focus application built with Tauri, Rust, and Svelte. The product combines Pomodoro sessions, task management, sync integrations, and system-level domain blocking and monitoring.

The repository currently starts with documentation so the project can move into implementation with clear product and technical boundaries.
